# Kitty Cross Road

`Kitty Cross Road` is a Java Swing arcade game inspired by lane-crossing games like Crossy Road.  
Pick your cat, dodge cars and trains, avoid drowning in water, and keep moving forward to increase your score.

## Features

- Three playable cats: Black, White, and Orange.
- Randomly generated lanes (road, train tracks, land, and water).
- Moving hazards:
  - Cars on road lanes.
  - Trains on track lanes.
  - Water strips where logs create safe paths.
- In-game menu with continue, restart, controls, and exit options.
- Sound effects and looping background music.

## Controls

- `Arrow Up`: Move forward (increases score if you progress).
- `Arrow Down`: Move backward.
- `Arrow Left`: Move left.
- `Arrow Right`: Move right.
- `Esc`: Pause/Resume and open the in-game pause menu.

## How Scoring Works

- Score is based on your farthest forward progress.
- Moving farther forward increases your score.
- Game over occurs when you:
  - Get hit by a car.
  - Get hit by a train.
  - Fall into water.
  - Move too far outside vertical bounds.

## Project Structure

- `src/it123`: Main Java source code.
- `Logos`: Image assets used by the game UI and sprites.
- `Sounds`: Audio files for music and SFX.
- `bin`: Compiled classes (generated by Eclipse / build).

## Requirements

- Java 15 or newer (project classpath is configured for JavaSE-15).
- A desktop environment that supports Java Swing.

## Run in Eclipse (recommended)

1. Open Eclipse.
2. Import the project:
   - `File` -> `Import` -> `Existing Projects into Workspace`.
   - Select the `KittyCrossRoad` folder.
3. Ensure the JRE is Java 15+.
4. Run `it123.Main`.
